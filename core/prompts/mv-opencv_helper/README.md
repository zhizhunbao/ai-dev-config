# OpenCV Code Helper

## 用途

帮助学习和使用 OpenCV 进行机器视觉任务，包括：

- 选择合适的 OpenCV 函数
- 理解和调优参数
- 编写完整可运行的代码
- 避免常见错误

## 使用场景

### 1. 函数选择

**输入：** "我想检测图像中的圆形物体"
**输出：** 推荐 `cv2.HoughCircles()`，解释原理，提供代码示例

### 2. 参数调优

**输入：** "Canny 边缘检测结果太多噪点"
**输出：** 建议增加阈值，调整高斯模糊参数，提供调优策略

### 3. 代码调试

**输入：** "我的轮廓检测代码找不到物体"
**输出：** 检查预处理步骤，建议添加形态学操作，提供改进代码

### 4. 性能优化

**输入：** "实时视频处理太慢"
**输出：** 建议缩小图像尺寸，使用更快的算法，提供优化方案

## 变量说明

- `task_description`: 要完成的视觉任务
- `image_characteristics`: 图像特征（噪声、对比度、分辨率等）
- `performance_requirement`: 性能要求（实时、高精度、平衡）
- `existing_code`: 现有代码（可选，用于改进或调试）

## 示例

### 基础使用

```
任务：去除图像噪声
图像特征：椒盐噪声
性能要求：平衡
```

### 高级使用

```
任务：检测并跟踪移动物体
图像特征：室内监控视频，光照变化
```
