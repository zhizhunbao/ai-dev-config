## INVEST 检查清单

用户故事应满足 INVEST 原则：

- [ ] **I**ndependent - 故事相互独立，可单独交付
- [ ] **N**egotiable - 细节可与团队协商调整
- [ ] **V**aluable - 为用户/业务提供明确价值
- [ ] **E**stimable - 团队可以估算工作量
- [ ] **S**mall - 足够小，可在一个 Sprint 内完成
- [ ] **T**estable - 有明确的验收标准可测试
